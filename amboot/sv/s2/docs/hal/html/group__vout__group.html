Set the clock source for vout.

Parameters:
[in]amb_hal_base_addressVirtual address where ambhal is loaded by OS.
[in]amb_clock_sourcethe new clock source.
[in]amb_clock_frequencythe clock frequency of the new source.
Note:
The amb_clock_frequency only needs to be specified for the clock sources AMB_PLL_REFERENCE_CLOCK_SOURCE_CLK_SI and AMB_PLL_REFERENCE_CLOCK_SOURCE_LVDS_IDSP_SCLK. Specify an amb_clock_frequency of 0 for all other clock sources. The topic Changing Clock Sources covers this in more details.
Return values:
AMB_HAL_SUCCESSa new clock source has been set.
AMB_HAL_FAILthe new clock source is not supported.

AMB_HAL_RETRYthe pll has not locked yet. try again.
