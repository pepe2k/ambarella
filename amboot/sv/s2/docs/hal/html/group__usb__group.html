enum amb_usb_port_state_t

USB Port State Settings.

Enumerator:
AMB_USB_OFF 

Disable USB port.

AMB_USB_ON 

Enable USB port.

AMB_USB_SUSPEND 

Force USB port into suspend state.

AMB_USB_ALWAYS_ON 

Enable USB port & force USB to never suspend.
